Login and Registration System (C++)

A simple Login and Registration System developed in C++ using Object-Oriented Programming (OOP) and File Handling.

Features

- User Registration
- User Login Authentication
- Username Validation
- Duplicate Username Detection
- Password Validation
- File-Based Credential Storage
- Success and Error Messages

Concepts Used

- Object-Oriented Programming (OOP)
- Classes & Objects
- Encapsulation
- Functions
- File Handling ("ifstream", "ofstream")
- Input Validation
- String Manipulation

Technologies

- C++
- Visual Studio Code / Code::Blocks
- GCC Compiler

How to Run

1. Clone this repository.
2. Compile the source code using a C++ compiler.
3. Run the executable.
4. Register a new user and log in using the created credentials.

Learning Outcome

This project enhanced my understanding of user authentication, file handling, and implementing OOP concepts in C++ by building a practical console-based application.
